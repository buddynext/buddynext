📋 Admin Members — Spec Notes
Access & Routing
• Page: wp-admin → BuddyNext → Members
• Capability: manage_options (admin only)
• Menu registered via add_submenu_page()
• URL: wp-admin/admin.php?page=buddynext-members
• Nonce verified on every action request
Data Sources
• Member list: WP users table + bn_* meta
• Post counts from bn_posts (not WP post_count)
• Trust level / role from usermeta _bn_role
• Last active from usermeta _bn_last_active
• Stats cards built via direct DB COUNT() queries
Actions & Bulk
• Bulk actions use WP nonces + Action Scheduler
• Large exports queued; download link emailed to admin
• Suspend = sets _bn_suspended usermeta + revokes auth
• Delete = soft-delete; 30-day recovery window
• Export CSV: name, email, role, join date, post count, last active
