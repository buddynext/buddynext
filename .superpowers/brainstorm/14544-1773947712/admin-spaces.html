📋 Admin: Spaces — Spec Notes
Page & Access
• Page: wp-admin → BuddyNext → Spaces
• Capability: manage_options
• Create space: admin can set privacy, moderators, linked forum/media
• Pending approval: spaces created by members when admin approval is required
• Archive = soft-delete; space hidden from directory but accessible via direct URL
Integrations Column
• JT badge (purple) = Jetonomy forum linked to this space
• MVS badge (teal) = WPMediaVerse linked to this space
• Admin can override linked forum from here (same as Space Settings, admin-wide)
• Linking creates a persistent association stored in space meta
• Multiple integrations per space supported
REST API
• GET buddynext/v1/spaces — list all (admin bypasses member permission checks)
• POST buddynext/v1/spaces — create
• PATCH buddynext/v1/spaces/{id} — update
• DELETE buddynext/v1/spaces/{id} — archive (soft delete)
• Filter params: category, privacy, status, search, per_page, page
