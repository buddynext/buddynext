📋 Hashtag Feed — Spec Notes
Routing & Templates
• URL: /community/tag/:slug/
• Template: templates/views/hashtag-feed.php
• Overridable via theme/buddynext/hashtag-feed.php
• Slug sanitized via sanitize_title() on creation
• 404 if slug not found in bn_search_index
REST & Data
• GET buddynext/v1/feed?hashtag=ux&sort=top&cursor=X
• sort options: top | latest (created_at DESC)
• Hashtag follow stored in bn_notification_prefs (type=hashtag)
• Hashtag counts from bn_search_index (denormalized)
• Counts recalculated via background job on post save
People Tab
• "People" tab: member directory filtered by top hashtag users
• Ranked by post count for this hashtag (last 90 days)
• Uses GET buddynext/v1/members?hashtag=ux&sort=posts
• Shows member card with follow button + post count
• Same member card component as /community/members/
