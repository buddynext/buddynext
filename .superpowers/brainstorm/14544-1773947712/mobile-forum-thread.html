Best practices for TypeScript generics? Struggling with conditional types
AJ
Alex Johnson 45 minutes ago Moderator
I've been working on a utility type library and keep running into issues with conditional types. Specifically, distributive conditional types are tripping me up. Anyone have good patterns for this? Here's a simplified example of what I'm trying to do...
type IsArray<T> = T extends any[] ? true : false;

Great question! The key is understanding distributive conditional types vs mapped types. For your case, you want to use [T] extends [any[]] instead of T extends any[] to avoid distribution. Here's why...
type IsArray<T> = [T] extends [any[]] ? true : false;
