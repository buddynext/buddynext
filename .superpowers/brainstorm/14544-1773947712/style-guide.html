Design System Decisions
Typography
• Inter for all body + UI text
• Plus Jakarta Sans for display/headings only
• 15px base (age-accessible)
• 1.7 line-height for reading comfort
• 4-size scale: xs(11) sm(13) base(15) lg(17)
Colour
• Warm grey neutrals (Notion palette)
• #0073aa brand (Wbcom ecosystem)
• #5b21b6 Jetonomy (forums)
• #0f766e WPMediaVerse (media)
• Full dark mode via CSS custom props
Frontend Stack
• PHP templates (theme-overridable)
• WP Interactivity API (reactivity)
• SSE for real-time counters
• buddynext/v1 REST API (data)
• Same pattern as Jetonomy + WPMediaVerse
