📋 Message Requests — Spec Notes
Who ends up here
• Users who don't follow you + you don't follow them
• Determined by user's DM access setting (everyone/followers/mutual/nobody)
• Connections (accepted) always go to main inbox
Actions
• Accept → conversation moves to main inbox
• Delete → removes request, sender not notified
• Block → adds to bn_blocks, can never message again
• Senders can't see read receipts until accepted
