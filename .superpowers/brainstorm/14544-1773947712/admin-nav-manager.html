🔌 How to register a custom tab from your plugin:
add_filter( 'buddynext_main_nav_items', function( $items ) { $items[] = [ 'id' => 'my-tab', 'label' => __( 'My Tab', 'my-plugin' ), 'icon' => '⭐', 'url' => '/my-page/', 'position' => 60, 'capability' => 'read', ]; return $items; } );
Same filter pattern for profile tabs (buddynext_profile_tabs) and space tabs (buddynext_space_tabs).
