📋 Forum Listing — Jetonomy Integration
Routing
• URL: /community/spaces/{slug}/forum/
• Template: BuddyNext registers route, renders Jetonomy's forum-listing template
• Theme-overridable: theme/jetonomy/forum-listing.php
• REST: GET jetonomy/v1/forums/{space_id}/threads?sort=top&category=all&cursor=…
Thread Mechanics
• Voting: POST jetonomy/v1/threads/{id}/vote {direction: up|down} — requires login
• Pinned: space admin can pin up to 3 threads (shown first regardless of sort)
• Hot algorithm: (replies × 2 + views × 0.5) / hours_since_post^1.5
• Answered flag: set when OP marks a reply as accepted answer
• Anonymous view: guests can read threads but not vote or reply
Permissions
• Post thread: requires space membership + post permission
• Reply: requires space membership
• Pin/lock/delete: space moderator or site admin only
• Vote: any logged-in user (not just space members)
