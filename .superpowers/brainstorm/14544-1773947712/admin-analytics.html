📋 Admin: Analytics — Spec Notes
Page & Pro Gate
• Page: wp-admin → BuddyNext → Analytics
• Pro feature — License::can_use_feature('analytics')
• Shows upgrade prompt / locked UI if on free plan
• Date range filter applies to all widgets on the page
• Export: CSV of raw events or aggregated summary
• GDPR: events stored with user_id; users can request deletion
Data Layer
• Table: bn_analytics_events
• Columns: event_type, user_id, object_id, object_type, timestamp, meta JSON
• Events: page_view, post_created, comment_added, reaction_given, space_joined, member_registered
• Aggregations run nightly via WP-Cron into summary tables for fast dashboard loading
• Heatmap: hourly bucketed event counts per day-of-week (7×24 matrix)
REST API
• GET buddynext/v1/analytics/summary?range=30d
• GET buddynext/v1/analytics/growth?range=12m (member counts by month)
• GET buddynext/v1/analytics/heatmap (7×24 activity matrix)
• GET buddynext/v1/analytics/top-spaces
• GET buddynext/v1/analytics/top-contributors
• All admin endpoints require manage_options capability
