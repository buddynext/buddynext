📋 Edit Profile — Spec Notes
Routing & Template
• URL: /community/u/:login/edit/
• Template: templates/views/edit-profile.php
• Auth: must be own profile or admin — redirect otherwise
• WP Interactivity API state: store('buddynext-profile', {…})
• Cover & avatar: REST POST buddynext/v1/profile/avatar
• Stored in WP media library, linked via user meta
Custom Field System
• Field types (from spec 05): text, textarea, url, social, select, multiselect, date, checkbox, number
• Each field: label, placeholder, required, visibility (public/followers/private)
• Fields registered via bn_register_profile_field() API
• Stored as wp_usermeta keys: bn_field_{field_id}
• Sections reorderable by admin in Settings → Profile Fields
Account & Saving
• Email/password use standard WP profile handler (wp_update_user)
• Notification preferences: separate settings screen (/edit/notifications/)
• Auto-save on blur (debounced 800ms) — "All changes saved" feedback
• Explicit "Save Changes" button also triggers full save
• Unsaved changes warning on navigation away
