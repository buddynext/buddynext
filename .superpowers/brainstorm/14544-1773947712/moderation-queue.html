📋 Moderation Queue — Spec Notes
Report Types
• Posts, comments, profiles, DMs (private)
• DM reports: admin sees only metadata, not content (privacy)
• Auto-escalate on 3+ reports in 1h
Admin Actions
• Dismiss: mark as not a violation
• Remove content: hides post/comment
• Warn: sends warning notification
• Strike: adds to user's record (3 = auto-suspend)
• Suspend: blocks login + posts
Reporter Experience
• Reporter notified when action taken (no details)
• db: bn_reports, bn_moderation_log
• Space admins can moderate within their spaces
