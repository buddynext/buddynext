📋 DM List — Spec Notes
Tabs
• All — all active convos
• Unread — only convos with unread messages
• Requests — unknown senders pending acceptance
• Archived — hidden but accessible
Conversation Actions (long-press/swipe)
• Pin / Unpin
• Mute / Unmute (with duration)
• Archive
• Delete (clears for you, not recipient)
• Report
Access Control
• Rate limit: 10 new convos/hour
• Blocked users can't appear in list
• Unknown senders → Requests tab
• REST: GET buddynext/v1/conversations
