Classic Sidebar Widgets — Appearance → Widgets
For themes with traditional widget areas. Drag into sidebars, footers, and anywhere widgets are supported.
👤 My Profile Widget
Shows logged-in user's avatar, name, and quick stats. Shows login form for guests.
Config: Title, stats to show (posts / followers / points), redirect on login
📰 Recent Community Activity
Latest posts, forum replies, and profile updates. Great for blog sidebars.
Config: Count (default 5), types to include, link to full feed
🏆 Top Members
Ranked member list by points or activity. Drives engagement visibility.
Config: Count, time period (all-time / month / week), link to leaderboard
🏘️ Active Spaces
Most active spaces by post count. Helps visitors discover community spaces.
Config: Count, sort (active / newest), show member count
👥 Who to Follow
Suggested members. Personalized for logged-in users, popular members for guests.
Config: Count, show mutual connections indicator
🔔 Notification Count
Bell icon with unread count. Tiny widget, ideal for header widget areas.
Config: Icon style, label text
💬 Latest Forum Threads
Recent Jetonomy threads. Good for blog sidebars to show community activity. Requires Jetonomy.
Config: Count, space filter, show vote count
🚀 Join Community Banner
Full CTA banner for guests. Hides for logged-in members. High conversion widget.
Config: Headline, description, button text, background color
📊 Community Stats Counter
Member count + active today + posts this week. Social proof for landing pages.
Config: Which stats to show, label text overrides
Shortcodes — Classic Editor & Page Builders
Works in Classic Editor, Divi, Elementor, Beaver Builder, and any context that renders shortcodes.
Shortcode Attributes Example
[buddynext_feed] type all / following / space  ·  count  ·  space_id  ·  hashtag [buddynext_feed type="following" count="10"]
[buddynext_members] count  ·  space_id  ·  role  ·  cols 2 / 3 / 4 [buddynext_members count="6" cols="3"]
[buddynext_spaces] count  ·  category  ·  cols [buddynext_spaces count="4" category="learning"]
[buddynext_profile] user_id blank = current  ·  style card / full [buddynext_profile style="card"]
[buddynext_leaderboard] count  ·  period all / month / week [buddynext_leaderboard count="5" period="month"]
[buddynext_stats] show members / posts / spaces [buddynext_stats show="members,posts"]
[buddynext_join_cta] headline  ·  button_text  ·  style banner / card / minimal [buddynext_join_cta style="banner" headline="Join our community"]
[buddynext_thread] thread_id  ·  show_replies yes / no [buddynext_thread thread_id="42"]
[buddynext_search] placeholder  ·  scope all / members / posts [buddynext_search scope="all"]
