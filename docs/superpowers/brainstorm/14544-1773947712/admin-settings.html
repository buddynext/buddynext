📋 Admin Settings — Spec Notes
Tabs
• General: registration, DM defaults, feed settings
• Integrations: addon toggles, per-addon config
• Webhooks: outgoing hooks with HMAC secret
• Roles: community roles + ability grants
• Email: from name/address, test email
• API: access token management
Access
• WordPress admin_menu, capability manage_options
• All BuddyNext menu items under wp-admin
• Each tab = separate page/section
Webhooks
• POST buddynext/v1/webhook/access (inbound)
• Outgoing webhooks via admin
• HMAC-SHA256 signature on all outgoing
• Events: post.created, user.registered, etc.
