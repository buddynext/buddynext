📋 Space Settings — Spec Notes
Tabs
• General: name, icon, cover, description, category, tags
• Privacy: visibility, post permissions, approval
• Members: role management, bans, invites
• Moderation: word blocklist, report settings
• Integrations: Jetonomy forum link, feed sync, MVS media tab
• Notifications: email/bell prefs for all members
Permissions
• Only space admin/moderator can access Settings
• Site admin sees all spaces' settings
• REST: PUT buddynext/v1/spaces/{id}
