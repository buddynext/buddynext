📋 Home Feed — Spec Notes
Layout & WordPress Context
• BuddyX / BuddyX Pro / Reign provide header + footer
• BuddyNext registers a secondary menu location below theme header
• Page content: max-width 1160px, two-column (feed + 300px sidebar)
• Template: templates/views/home-feed.php (overridable via theme/buddynext/)
• Auth redirect: unauthenticated users → login page
Feed Card Types
• BuddyNext native post — text, photo, link preview, poll, file
• Events addon (separate plugin) — event cards injected via feed hook
• Jetonomy forum card — purple left border, vote controls, opens in forum
• WPMediaVerse media card — teal left border, photo/video grid
• Toggled via Space settings: "Push forum activity to feed" (off by default)
• Toggled via Space settings: "Push media to feed" (on by default)
Frontend Architecture
• PHP templates + WP Interactivity API (no build step)
• store('buddynext', { state, actions }) — same pattern as Jetonomy
• SSE endpoint: /wp-json/buddynext/v1/sse — live notif + DM badge counts
• REST: GET buddynext/v1/feed?scope=home&cursor=X (cursor pagination)
• Dark mode: CSS custom properties, toggle stores pref in localStorage
Sidebar Widgets
• "People to Follow" — friend-of-friend + follows-you signals
• "Trending Discussions" — Jetonomy integration, top posts last 24h
• "Active Spaces" — joined spaces with unread count
• "Featured Media" — WPMediaVerse integration, curated/recent
• All widgets hidden for guests; show prompts to join
Design System
• Font: Inter (body/UI) + Plus Jakarta Sans (display/headings)
• Base size: 15px, line-height: 1.7 (any-age readable)
• Colours: Notion warm greys (--text-1 #37352f)
• Brand: #0073aa / Dark: #4dabdb
• Integration accents: Jetonomy #5b21b6, WPMediaVerse #0f766e
Composer
• Click input → expands inline (no modal)
• Post types: text, photo(s), link preview, poll, file — all V1
• Audience: Public / Followers / Space (selector appears on expand)
• Max 4 images in 2×2 grid
• Character limit: 3000 with counter shown at 2500+
