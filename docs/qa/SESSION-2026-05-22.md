## Trap caught + codified

**Composer worktree autoload trap.** Sub-agents that ran `composer dump-autoload` inside `/tmp/bn-{branch}/` baked worktree paths into `vendor/composer/autoload_static.php`. Every class load broke with `class "BuddyNext\Core\Plugin" not found` after the worktree was removed. Fix: `composer dump-autoload -o` from canonical repo. Prevention: every dispatched agent prompt now includes the explicit warning.

## Agent overclaim pattern

Sub-agents sometimes report "already shipped before this branch" for features that exist in code but hide behind conditional renders (`if ($total_unread > 0)`). Live walks on sparse dev data return false-positive gap reports. Cross-check: grep the template first; if found, check the conditional; if guard is correct, seed realistic test data.
