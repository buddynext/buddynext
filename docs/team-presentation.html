Plugin Ecosystem
Six plugins. One platform.

Plugin Roles

BuddyNext

The social + community hub. Owns bn_* tables, social graph, activity feed, spaces, notifications, member profiles. UI layer for everything.

WPMediaVerse

Media engine AND DM engine. Owns all mvs_* tables. Standalone or community mode — same tables, same code, no migration needed.

Jetonomy

Forums and threaded discussions. Defers social graph to BuddyNext when active. Forum tab appears inside BuddyNext spaces.

WBGamification · Career Board

Addons that hook into BuddyNext actions. Award points on post_created, follow, join. Jobs appear as cards in the activity feed.

Key Architecture Decision — DM
WPMediaVerse owns DM. BuddyNext is the UI layer.

Single source of truth. One set of tables (mvs_*), one codebase. WordPress wp_users is the same table in standalone and community mode — no user mapping, no migration, no dependency needed.

Free
1:1 DM works for all BuddyNext Free users if WPMediaVerse Free is active. DM tab hidden with a notice if WPMediaVerse not installed.
Pro
WPMediaVerse Pro is bundled in BuddyNext Pro — every Pro user automatically gets Group DM, WebSocket real-time, and read receipts. No separate purchase.
Bridge
3 hooks only: set mvs_buddynext_active → true, inject bn_blocks check, route message notifications to bn_notifications.
Integration Pattern
How addons plug into BuddyNext

BuddyNext never calls addon code directly. Every integration is hooks.

BuddyNext Action Who hooks it What they do
buddynext_post_created WBGamification Award points for posting
buddynext_follow WBGamification Award points for following
buddynext_space_member_joined WBGamification Award points for joining a space
mvs_media_uploaded BuddyNext bridge Creates bn_posts entry (type: media) + fires buddynext_index_hashtags
mvs_message_sent BuddyNext bridge Creates bn_notifications entry (type: bn.new_message)
jetonomy_discussion_created BuddyNext bridge Creates bn_posts entry (type: discussion) + indexes hashtags
buddynext_space_tabs (filter) Jetonomy bridge Injects Forum tab into linked BuddyNext spaces
buddynext_index_hashtags BuddyNext core Extracts #tags from content, populates bn_post_hashtags
BuddyNext fires actions Addons hook in BuddyNext bridges fire for each addon No direct addon-to-addon calls
