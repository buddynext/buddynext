A modern social fabric for WordPress, redrawn from first principles.

Twenty-eight redesigned screens powered by a single token system: OKLCH-derived color, density that flexes from compact to spacious, and live-tweakable accent + radius. Open any screen and use the ⚙ Tweaks floating button to retune the system in real time.
