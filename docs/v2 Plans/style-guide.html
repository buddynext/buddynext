Design System · v2 · 2026

A community interface that ages like wood, not plastic.

Theme-independent. White-label per community via a single hue token. WCAG AAA defaults. Built for the next five years of community work — AI, real-time presence, threaded conversations, voice rooms, paid spaces, federation.

01 · FOUNDATION

Eight rules that drive every choice

If a decision can be made one of two ways, this list breaks the tie.

01
Theme-independent
No theme contracts in the plugin. Renders identically on Twenty-Twenty-Four, BuddyX, Reign — or fully headless.
02
One hue, infinite brands
Communities rebrand by overriding --bn-hue. Everything else is derived in oklch.
03
Density is a setting
Comfortable for new members; compact for power users. Same components, different scale.
04
Mobile parity
Designed as if there's a native app, because there will be. 44px hit targets everywhere.
05
AAA contrast as default
Large-text mode and dyslexia mode are first-class, not afterthoughts.
06
AI is a tool, not the product
Composer help, summaries, semantic search. Always opt-in, always disclosed, always editable.
07
Real-time without performance
Presence dots, live cursors, typing indicators — quiet and ambient, never blinking demands.
08
Federated by design
Posts can carry an external source. Cross-community feeds are a first-class layout, not a bolt-on.
